基础栈

Python 3.12

FastAPI

Pydantic v2

pydantic-settings 管配置

SQLAlchemy 2.x async

Alembic

PostgreSQL

Redis

对象存储：阿里 OSS / 腾讯 COS / AWS S3 三选一

异步任务：Celery 或 ARQ

部署：Docker + Nginx

Pydantic 官方当前文档展示的是 v2.12.5，配置管理推荐走 pydantic-settings；新项目直接按 v2 写最合适。

数据库层建议直接按 SQLAlchemy 2 的异步模式组织，官方文档提供了 AsyncSession / async_sessionmaker 示例；数据库迁移继续用 Alembic。

FastAPI 自带的 BackgroundTasks 更适合小任务；如果是短信、推送、支付补单、统计报表、订阅消息发送这类重任务，官方文档也明确更适合放到 Celery 这类独立任务队列里。
